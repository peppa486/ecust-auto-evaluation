# 华东理工大学教务系统一键评教助手 (ECUST Auto Evaluation)

本工具专为华东理工大学教务处统一评价系统（强智教务系统 `jsxsd`）开发。支持**多门课程一键批量半自动流式评教**，免安装任何浏览器插件，在 F12 控制台粘贴回车即可全自动/半自动流式完成。

---

## 🎁 压缩包内容说明
* `console-batch-evaluation.js`：多门课程批量评教核心代码（在待评课程列表页使用）。
* `README.md`：本使用指南。

---

## 🚀 使用方法

### 🛠️ 场景：多门课程智能批量评教
当您点击“学生评教”进去后，会看到您所有待评课程的列表页（操作列为“进入评价”）。

1. 在此**课程列表页面**，按下键盘上的 **`F12`** 键（或右键网页空白处选择“检查”），在弹出的控制面板顶部切换到 **`Console`（控制台）**。
2. 双击打开本目录下的 `console-batch-evaluation.js`，复制其全部代码，粘贴到控制台的光标处。
3. 按下 **`Enter`（回车键）** 运行。
4. **流式操作体验**：
   * 页面底部会动态弹出一个评教子窗口，**自动加载第 1 门课程**并**瞬间填好所有单选框和文字评语**。
   * 您只需在底部网页中，手动点击 **`[提交]`** 并确认学校系统的保存弹窗。
   * 提交完成后，助手会在后台自动感知，**瞬间加载第 2 门课程并填好**，继续等待您点击提交。
   * 依次类推，直到所有课程全部评完，会自动弹出“恭喜！所有课程评教已自动完成！”提示，并自动刷新当前列表状态。

> 💡 **防雷同与规则适配说明**：
> * **防雷同机制**：脚本会根据设定的优秀率（默认95%），随机产生一定比例的“9分（良好）”分值以模拟人类真实的打分波动，防止系统筛查雷同满分。
> * **最爱课程限额**：学校后台规定全校至多评选 3 个“最喜爱的课程”。本脚本包含全局计数，在评满 3 个后会自动将后续课程选为“否”，彻底避开教务处后台的报错拦截。
