# 华理教务一键评教油猴脚本设计规格书 (ECUST Auto Evaluation Spec)

## 1. 概述
本设计文档旨在为华东理工大学教务系统（强智教务系统 `jsxsd`，域名 `inquiry.ecust.edu.cn`）开发一个一键评教 Tampermonkey（油猴）脚本。
用户安装此脚本后，将在评教表单页面右侧看到一个精美的悬浮控制面板。通过该面板，用户可以一键完成所有评分单选框的精确填充、是否最爱课程的自动选择，以及自动生成得体的教师评语。脚本支持“优秀率”的精细配置和防秒投延迟提交机制。

## 2. 功能需求

### 2.1 UI 悬浮面板 (Floating Panel)
* **样式**：采用现代**毛玻璃模糊质感（Glassmorphism）**，圆角卡片设计。
* **主题色**：华东理工大学标准蓝（`#004F9F`）作为强调色。
* **布局**：
  * 面板标题：“ECUST 评教助手”
  * 核心设置区：
    * “优秀率”滑动条（80% - 100% 调节，默认 95%）
    * “自动填评语”开关（默认开启）
  * 操作按钮区：
    * 【一键填充（安全）】：勾选所有单选框并填入评语，不提交。
    * 【一键提交（快捷）】：填充后，延迟 1.5 秒自动提交表单。
  * 状态提示区：显示当前识别到的指标数和操作状态。
  * 最小化按钮：可收缩为一个精致的华理蓝色圆形浮标，再次点击展开。

### 2.2 评分项精准填充算法
根据华理教务系统的 DOM 结构分析：
1. **指标项（单选框组）填充**：
   * 页面共包含 12 项指标（隐藏域为 `pj06xh`，其 value 是 1 到 12）。
   * 每项指标的单选框 `name` 为：`pj0601id_X`（`X` 为对应指标序号）。
   * 每一组单选框有 11 个单选项（对应分值 10, 9, 8, ..., 0）。
   * **比例填充逻辑**：
     * 读取优秀率 $P_{opt}$（例如 95%）。
     * 对第 $X$ 项指标：
       * 产生随机数 $R$。
       * 若 $R < P_{opt}$，则勾选该组第 1 个单选框（即分值 10，最高分）。
       * 若 $R \ge P_{opt}$，则勾选该组第 2 个单选框（即分值 9，良好分）。
       * 这样能在保障最终高分评教的同时，产生极其真实的随机波动。
2. **最爱课程单选框填充**：
   * 页面底部的“是否最喜欢的课程”单选框：`name="sfzxhkc"`。
   * 脚本默认勾选 `value="1"`（是）的单选项。

### 2.3 评语自动生成
* **定位输入框**：
  * 定位建议内容文本域 `id="jynr"`。
* **评语生成**：
  * 从内置的 10 条华理评教语库中，随机挑选一条正面评语（字数 > 15字，符合学校字数限制）并填入。
  * 评语库示例：
    1. 老师授课条理清晰，讲解生动，非常注重与我们的课堂互动，收获颇丰。
    2. 教学态度十分认真负责，课后答疑非常有耐心，对我们的学习帮助很大。
    3. 老师备课充分，重难点突出，讲课深入浅出，是一位非常敬业的好老师。
    4. 课堂教学方法灵活，课外作业设计适中，反馈很及时，对知识开拓很有帮助。
    5. 老师很有亲和力，讲课内容丰富，课堂氛围非常活跃，极大地激发了学习兴趣。

### 2.4 防检测安全机制
* **模拟点击**：
  * 必须对目标 radio 节点显式触发 `.click()` 事件，而不是仅修改 `.checked = true`，确保能完美触发教务系统自带的隐藏得分关联域修改事件。
* **模拟延迟与防秒提**：
  * 按钮【一键提交】执行后：
    * 先进行【填充】。
    * 延迟 1.5 秒后，自动调用网页原生的 `saveData(document.getElementById('tj'), '1')` 函数触发正式提交，并触发系统的保存确认。

## 3. 技术设计
* **匹配网址**：`https://inquiry.ecust.edu.cn/jsxsd/xspj/xspj*` 
* **语言**：原生 JavaScript，无外部依赖。
* **拖拽支持**：实现悬浮窗在页面中的鼠标自由拖拽。
